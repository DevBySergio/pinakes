# Sample Pinake Workspace

This directory is a minimal Pinake workspace for documentation authors and automation maintainers.

It demonstrates:

- `.pinake/pinake.json` as the manifest and document registry;
- `.pinake/docs` as the Markdown content root;
- `.pinake/.gitignore` ignoring generated `.state` files.

Open this folder in VS Code with Pinake Editor installed, then run `Pinake: Validate`.
